Features

Everything you need to
be anywhere.

All the power of pricier tools like iMyFone AnyTo and Tenorshare iAnyGo — on iPhone, Android, Mac & Windows, with a real free trial.

One-tap Teleport

Drop a pin anywhere on the map — a landmark across the world or the coffee shop down the street — and you're there instantly. Search by name, paste coordinates, or tap the map.

Joystick Walk

Move your location in real time with an on-screen joystick. Push to steer, farther to go faster.

Route Mode

Drive a path with realistic speed, stops, acceleration and a live ETA — like you're really driving it.

PoGo Mode

Jump-teleport built for Pokémon GO, with a cooldown timer so you can hop between spots without watching the clock yourself.

GPX Import

Bring your own paths — import a GPX file and Wander walks or drives the whole track with realistic speed.

Natural Drift

Optional GPS jitter adds tiny realistic movement, so your position never looks robotically frozen.

Saved Places

Bookmark favorite spots and recent locations, then jump back with a single tap from the Places tab.

Wi-Fi or Cellular

No Wi-Fi? Flip on Airplane Mode and Wander keeps working on cellular — spoof from anywhere.

Runs Untethered

After setup, no computer needed. Wander re-signs and refreshes itself so it never expires on you.

Auto-Updates

New versions download and install themselves over the air — you're always on the latest build.
